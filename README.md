# 🗣️ VaaniAI — AI-Powered Citizen Grievance System

A web application that allows citizens to submit complaints which are automatically categorized and prioritized using Groq AI.

## 🚀 Live Demo
[Click here to view live](https://heartfelt-bunny-408540.netlify.app)

## ✨ Features
- 🤖 AI automatically categorizes complaints (Road, Water, Electricity, etc.)
- 🔴 Priority detection (High, Medium, Low)
- 📊 Real-time dashboard with analytics
- 🔍 Search and filter complaints
- 🔒 Admin panel with password protection
- 📱 Mobile responsive
- 📥 Export complaints to Excel/CSV
- 🔄 Real-time sync across all devices (Firebase)

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Groq AI API (LLaMA 3.3)
- Firebase Realtime Database
- Netlify (Deployment)
